<header>:
Utilizada para representar a introdução de uma seção ou uma seção de cabeçalho.
Conteúdo típico inclui logotipos, títulos, sub-títulos, etc.

<nav>:
Usada para representar uma seção de navegação.

<main>:
Define o conteúdo principal da página. Deve ser exclusivo e não aninhado dentro de outras tags <main>.

<article>:
Utilizada para representar uma composição autônoma, que pode ser distribuída e reutilizada independentemente.

<section>:
Usada para agrupar conteúdo temático dentro de uma página.

<aside>:
Representa conteúdo que está relacionado ao conteúdo ao redor, mas pode ser considerado separado dele.

<footer>:
Utilizada para representar o rodapé de uma seção ou da página.
